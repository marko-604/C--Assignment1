\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_critter}{Critter} class and its derived classes, as well as the \doxylink{class_critter_wave_generator}{Critter\+Wave\+Generator}. 

This file defines the base class for critters in the \doxylink{class_tower}{Tower} Defense game along with specialized critters (\doxylink{class_squirrel}{Squirrel}, \doxylink{class_wolf}{Wolf}, \doxylink{class_bear}{Bear}). It also declares the \doxylink{class_critter_wave_generator}{Critter\+Wave\+Generator} for generating waves of critters.
