\doxysubsection{Detailed Description}
Generates waves of critters for the game. 

The \doxylink{class_critter_wave_generator}{Critter\+Wave\+Generator} class creates a vector of \doxylink{class_critter}{Critter} pointers based on the current wave number. The wave number can be incremented to scale the difficulty of subsequent waves.

Generates a new wave of critters. 

Generates a wave of critters based on the current wave number.

The method creates and returns a vector of \doxylink{class_critter}{Critter} pointers. The number and type of critters generated may depend on the current wave number.

\begin{DoxyReturn}{Returns}
A vector containing pointers to the newly generated critters.
\end{DoxyReturn}
Creates a vector of \doxylink{class_critter}{Critter} pointers where the number and type of critters depend on the current wave number. For each iteration, different critter types (\doxylink{class_squirrel}{Squirrel}, \doxylink{class_wolf}{Wolf}, \doxylink{class_bear}{Bear}) are added based on the remainder when dividing the loop counter by 2, 3, or 5.
