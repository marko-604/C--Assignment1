\doxysubsection{Detailed Description}
Represents a moving enemy in the \doxylink{class_tower}{Tower} Defense game. 

A \doxylink{class_critter}{Critter} has attributes such as reward value, speed, health, strength, and position. It also has a unique identifier assigned automatically. Methods allow manipulation of its state, movement on the map, and checking its status.

void Critter\+::move (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{class_map}{Map}} \texorpdfstring{$\ast$}{*}}]{game\+Map}{}\end{DoxyParamCaption})}



Moves the critter on the provided game map. 

Moves the critter to an adjacent cell that brings it closer to the exit.

This function determines the next position for the critter based on its speed and the map layout.


\begin{DoxyParams}{Parameters}
{\em game\+Map} & Pointer to the \doxylink{class_map}{Map} object.\\
\hline
\end{DoxyParams}
The function attempts to move the critter one cell at a time by checking cells in the order left, right, up, and down. For each direction, it uses the map\textquotesingle{}s is\+Closer\+To\+Exit() and is\+Path\+Cell() functions to determine if the new position is valid. If a valid move is found, the critter\textquotesingle{}s current cell is reset to a path cell, the critter\textquotesingle{}s coordinates are updated, and the new cell is marked as a critter cell.
