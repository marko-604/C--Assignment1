\doxysubsection{Detailed Description}
Definitions of the \doxylink{class_tower}{Tower} classes used in the \doxylink{class_tower}{Tower} Defense game. 

This file declares the base \doxylink{class_tower}{Tower} class and several derived classes\+:
\begin{DoxyItemize}
\item \doxylink{class_sniper_tower}{Sniper\+Tower}\+: A tower with high damage and long range.
\item \doxylink{class_bomb_tower}{Bomb\+Tower}\+: A tower that inflicts area-\/of-\/effect damage.
\item \doxylink{class_freezing_tower}{Freezing\+Tower}\+: A tower that slows critters.
\end{DoxyItemize}

It also declares an external global vector, {\ttfamily game\+Critters}, which holds pointers to active critters in the game.

\begin{DoxyNote}{Note}
The implementation of these classes is provided in \doxylink{towers_8cpp}{towers.\+cpp}.
\end{DoxyNote}
