\doxysubsection{Detailed Description}
Represents a grid-\/based map used in the \doxylink{class_tower}{Tower} Defense game. 

The \doxylink{class_map}{Map} class maintains a 2D grid of cells. Each cell has a type (SCENERY, PATH, TOWER, or CRITTER) and flags to indicate if it is the entry or exit point. The class supports setting cell types, defining entry/exit points, validating the connectivity of the map using BFS, and displaying the map and the path from entry to exit.
