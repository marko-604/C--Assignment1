\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_map}{Map} class for the \doxylink{class_tower}{Tower} Defense game. 

This file declares the \doxylink{class_map}{Map} class that manages a grid-\/based game map. The map consists of cells that can be of various types (SCENERY, PATH, TOWER, CRITTER). It provides functions to configure the map (e.\+g., setting entry and exit points), validate its configuration, and display the map and a calculated entity path.
